# Helm Common Charts

This repository contains a collection of common Helm charts that can be used across different projects.

## Charts

### chart-common
A common chart template that can be used as a base for other charts.

## Usage

To use these charts, add the repository to your Helm:

```bash
helm repo add helm-common https://mano1233.github.io/helm-common
helm repo update
```
